Gotunnel

Share you localhost server or directory over the internet. supports HTTP/S and websocket.

View the Project on GitHub ciju/gotunnel

What can it do?

localhost server over internet.

Make a local server accessible over the internet. With HTTP/S and WebSocket support, your user/client can interact with the localhost site, as if its hosted on cloud.

Local directory shared over internet.

If you just want to share a directory with others, you just have to start Gotunnel with a few command line options (no need to run another server).

Limitations

You have to be online, and running the server and Gotunnel, for the site to be accessible :) Speed. Its going to be slower than a hosted system.
