Gotunnel

Share you localhost server or directory over the internet. Supports HTTP/S and websocket.

View the Project on GitHub ciju/gotunnel

What can it do?

  • Localhost server over internet.

    Make a local server accessible over the internet. With HTTP/S and WebSocket support, your user/client can interact with the localhost site, as if its hosted on cloud.

    Example, lets say the localhost server is running on port 3000. To make it accessible over the net, you would run.

    $ ./gotunnel -p 3000
Your site should be available at: abcd.localtunnel.net

    Your local site should now be accessible at abcd.localtunnel.net

  • Share local directory over internet.

    If you just want to share a directory with others, you just have to start Gotunnel with a few command line options (no need to run another server).

    To share the current directory, you would run

    $ ./gotunnel -fs -p <port>
Serving <your directory path> at port <port>
Your site should be available at: abcd.localtunnel.net

    The port is some arbitrary free port, where the directory server would run.

Installation

OS X

Download the binary.

Make it executable. For added convenience, you could add its diretory in $PATH. Ex:

$ chmod u+x gotunnel
$ mv gotunnel /usr/local/bin

Others

Download the project (as a zip/tar) or clone the repo. And then compile with golang, as follows.

  • First make sure golang is installed.
  • then run the following in the extracted directory. 
    $ go get
$ make
    You should now have a binary in the ./bin directory. Make it executable. And add it to the $PATH

Limitations

  • You have to be online, and running the server and Gotunnel, for the site to be accessible :)
  • Speed. Its going to be slower than a hosted system.
